# Front-End Web Technology Lab

## 📖 Overview

This repository contains practical programs, assignments, and lab exercises completed as part of the **Front-End Web Technology** course. The purpose of this repository is to practice modern web development concepts and build interactive, responsive, and user-friendly web applications.

## 🚀 Topics Covered

### 1. JavaScript

* Variables and Data Types
* Operators and Expressions
* Conditional Statements
* Loops
* Functions
* Arrays and Objects
* DOM Manipulation
* Event Handling

### 2. jQuery and ES6

* jQuery Selectors and Events
* Effects and Animations
* AJAX Basics
* ES6 Variables (let, const)
* Arrow Functions
* Template Literals
* Destructuring
* Spread and Rest Operators
* Modules and Classes

### 3. Introduction to ReactJS

* React Components
* JSX Syntax
* Props and State
* Functional Components
* Component Lifecycle Basics
* Creating Interactive User Interfaces

### 4. Routing, Hooks, and API Integration in ReactJS

* React Router
* Navigation Between Pages
* useState Hook
* useEffect Hook
* Custom Hooks
* Fetching Data from APIs
* Handling API Responses
* Error Handling and Loading States

### 5. Authentication and API Best Practices

* User Authentication Concepts
* Login and Registration Forms
* JWT Authentication
* Protected Routes
* Secure API Calls
* Environment Variables
* Error Handling
* Code Organization and Best Practices

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* jQuery
* ReactJS
* React Router
* REST APIs

## 🎯 Objective

The objective of this repository is to strengthen front-end development skills through hands-on practice and implementation of modern web technologies.

## 📂 Repository Structure

Each folder contains lab programs, examples, and assignments related to a specific topic covered in the course.
